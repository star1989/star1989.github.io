TensorFlow安装

2017/3/20 21:38 下午 posted in  ml  

决定你的tensorflow类型

选择一种最适合你的tensorflow进行安装:
* 只支持CPU.
* 不仅支持CPU,还支持GPU.需要硬件支持NVIDIA CUDA GPU

只有集显的我默默流泪,需要的朋友请自取NVIDIA's documentation

tensorflow的安装方式

  • virtualenv
  • "native" pip
  • Docker
  • from source
virtualenv(官方推荐)
  • 开启shell
  • 安装pip和virtualenv 
 $ sudo easy_install pip
 $ sudo pip install --upgrade virtualenv
  • 创建虚拟环境

$ virtualenv --system-site-packages targetDirectory

注:当前目录为~/tensorflow

  • 激活环境
 $ source ~/tensorflow/bin/activate      # If using bash, sh, ksh, or zsh
 $ source ~/tensorflow/bin/activate.csh  # If using csh or tcsh

正确显示如下:

 (tensorflow)$
  • 安装tensorflow(pip >= 8.1)
 $ pip install --upgrade tensorflow      # for Python 2.7
 $ pip3 install --upgrade tensorflow     # for Python 3.n
 $ pip install --upgrade tensorflow-gpu  # for Python 2.7 and GPU
 $ pip3 install --upgrade tensorflow-gpu # for Python 3.n and GPU
  • 修复激活失败(安装正确请跳过)
 $ pip install --upgrade TF_BINARY_URL   # Python 2.7
 $ pip3 install --upgrade TF_BINARY_URL  # Python 3.N
  • mac,python 3.4
 $ pip3 install --upgrade \
 https://storage.googleapis.com/tensorflow/mac/cpu/tensorflow-1.0.1-py3-none-any.whl

上面的url需要export到环境

使用
  • 选择使用的shell类型
$ source ~/tensorflow/bin/activate      # bash, sh, ksh, or zsh
$ source ~/tensorflow/bin/activate.csh  # csh or tcsh
  • 暂停tensorflow virtualenv

(tensorflow)$ deactivate

  • 删除tensorflow virtualenv

(tensorflow)$ deactivate

pip安装

tip:需要pip version >= 8.1

  • 升级pip命令
$ sudo easy_install --upgrade pip
$ sudo easy_install --upgrade six
  • 安装tensorflow
 $ pip install tensorflow      # Python 2.7; CPU support (no GPU support)
 $ pip3 install tensorflow     # Python 3.n; CPU support (no GPU support)
 $ pip install tensorflow-gpu  # Python 2.7;  GPU support
 $ pip3 install tensorflow-gpu # Python 3.n; GPU support
  • 卸载tensorflow
$ pip uninstall tensorflow
$ pip3 uninstall tensorflow
docker安装
  • 拉取镜像
docker pull gcr.io/tensorflow/tensorflow
docker pull gcr.io/tensorflow/tensorflow:latest-devel
