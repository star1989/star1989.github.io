使用python搭建ss梯子

CentOS7

yum install python-setuptools && easy_install pip
pip install shadowsocks

启动命令

ssserver -p 8836 -k `password` -m rc4-md5
#或者可以通过以下指令在后台启动shadowsocks的服务：
ssserver -p 8836 -k `password` -m rc4-md5 -d start
ssserver -p 8836 -k `password` -m rc4-md5 -d stop

优化版本
* 创建/etc/shadownsocks.json配置文件
* 写入配置:

{
    "server":"你的服务器ip地址",
    "server_port":10088,
    "local_address": "127.0.0.1",
    "local_port":1080,
    "password":"你设置的密码",
    "timeout":300,
    "method":"aes-256-cfb",
    "fast_open": false
}

优化后的启动命令:

ssserver -c /etc/shadowsocks.json
#或者在后台运行
ssserver -c /etc/shadowsocks.json -d start
ssserver -c /etc/shadowsocks.json -d stop

python语法总结

变量

命令规则

  • 必须是大小写英文,数字和_的组合,且不能用数字开头
  • 不用声明,直接使用(同PHP)

类型

  • 整数
  • 浮点
  • 字符串
  • 布尔
  • 空: None (与0不同)
  • Tuple : 元组,值不可变更
  • List : 表,值可以变更[下限:上限:步长]
  • Dictionary : 字典
  • 自定义数据类型

语法

缩进

python强制使用缩进来解析块代码.

运算符

  • 数学 : +,-,*,/,**,%
  • 判断 : ==,!=,>,>=,<,<=,in
  • 逻辑 : and,or,not

条件语句

if condition1:
    do1
elif condition2:
    do2
elif condition3:
    do3
else:
    do4

循环

for

for elemnt in list:
    statment

while

while condition:
    statement
