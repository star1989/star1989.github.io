python语法总结

变量

命令规则

  • 必须是大小写英文,数字和_的组合,且不能用数字开头
  • 不用声明,直接使用(同PHP)

类型

  • 整数
  • 浮点
  • 字符串
  • 布尔
  • 空: None (与0不同)
  • Tuple : 元组,值不可变更
  • List : 表,值可以变更[下限:上限:步长]
  • Dictionary : 字典
  • 自定义数据类型

语法

缩进

python强制使用缩进来解析块代码.

运算符

  • 数学 : +,-,*,/,**,%
  • 判断 : ==,!=,>,>=,<,<=,in
  • 逻辑 : and,or,not

条件语句

if condition1:
    do1
elif condition2:
    do2
elif condition3:
    do3
else:
    do4

循环

for

for elemnt in list:
    statment

while

while condition:
    statement

continue
break
