  • 参数默认值

可混用,混用原则:先位置,后关键字,包裹位置,包裹关键字

packing

>>> def pack(*param):
...     print param
>>> pack(1,2,3,4,"abc")
(1, 2, 3, 4, 'abc')

平常应用,可以强制将tuple转换成list:

>>> def pack2(*param):
...     param = list(param)
...     param[0] = "Hello"
...     param[1] = "Python"
...     print param
>>> pack2(1,2,"end")
['Hello', 'Python', 'end']

函数对象(lambda)

这种能够接受一个函数作为参数的函数叫做「高阶函数」（higher-order function），是来自函数式编程（functional programming）的思想。

>>> seq=[1,2,3,4,5]
>>> map(lambda x:x**3, [ x for x in seq ])
[1, 8, 27, 64, 125]

高阶函数:
* filter(function, sequence)：对sequence中的item依次执行function(item)，将执行结果为True的item组成一个List/String/Tuple（取决于sequence的类型）返回
* map(function, sequence) ：对sequence中的item依次执行function(item)，见执行结果组成一个List返回
* reduce(function, sequence, starting_value)：对sequence中的item顺序迭代调用function，如果有starting_value，还可以作为初始值调用，例如可以用来对List求和
* lambda：这是Python支持一种有趣的语法，它允许你快速定义单行的最小函数，类似与C语言中的宏，这些叫做lambda的函数，是从LISP借用来的，可以用在任何需要函数的地方

lambda与def的差别

map( lambda x: x*x, [y for y in range(10)] )

def sq(x):
    return x * x
map(sq, [y for y in range(10)])

面向对象的类

定义

使用class关键字定义一个类,并且类名首字母大写.

语法

class PyClass:
variables
functions

创建对象

myClass = PyClass()

引用对象属性

myClass.attribute

作用域

  • 通过self调用类属性
  • 对象生成时自动执行的特殊方法: __init__()
  • 类的属性是对数据的封装,方法则是对类的行为封装
  • 类属性:
    1. 公有属性.
    2. 内置属性:由系统在定义类的时候默认添加,由前后两个下划线构成__dict__,__module__
    3. 私有属性:不能被类以外的函数调用.定义时在属性名前加__
  • 方法也分为公有方法和私有方法
    1. 私有方法:以__定义
    2. 类方法:被classmethod()函数处理过的函数,添加类装饰器@classmethod
    3. 静态方法:等于"全局函数"
from functools import wraps

class A:
    # Decorator as an instance method
    def decorator1(self, func):
        @wraps(func)
        def wrapper(*args, **kwargs):
            print('Decorator 1')
            return func(*args, **kwargs)
        return wrapper

    # Decorator as a class method
    @classmethod
    def decorator2(cls, func):
        @wraps(func)
        def wrapper(*args, **kwargs):
            print('Decorator 2')
            return func(*args, **kwargs)
        return wrapper
