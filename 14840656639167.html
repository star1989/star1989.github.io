swoole安装

2017/1/11 0:27 上午 posted in  php  

下载源码

swoole oschina link

在mac下编译

编译swoole

cd swoole-src
phpize
./configure
make
sudo make install

添加扩展至php.ini

extension=/usr/local/php/lib/php/extensions/debug-zts-20131226/swoole.so
